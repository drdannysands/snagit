Screenshot with SnagIt

Since I can never remember keystrokes to do various things, I needed a keyword-triggered workflow to start a screen capture using SnagIt.

Requires SnagIt to be installed.

Set Environment Variable to the name of your version of SnagIt (default is "SnagIt 2023").
